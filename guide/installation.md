# Installation

## Install With ZIP File

请从 [中国下载地址](https://gitee.com/ZihaoVistonWang/Stata-AI-Skill/releases/download/v1.0/stata-ai-skill-v1.0.zip) 或 [国际下载地址](https://github.com/ZihaoVistonWang/Stata-AI-Skill/releases/download/v1.0/stata-ai-skill-v1.0.zip) 下载 `stata-ai-skill-v1.0.zip` 文件。然后，解压该 ZIP 文件，并将其中的 skills/stata-ai-skill 文件夹完整复制到你的 Skills 安装目录中。请保留文件夹内的原始结构，并在安装完成后读取 SKILL.md。

## Install With skills.sh

```bash/cmd
npx skills add ZihaoVistonWang/Stata-AI-Skill --skill stata-ai-skill
```
