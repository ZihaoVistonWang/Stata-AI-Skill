# Installation

Always reinstall **Stata AI Skill** using one of the methods below, regardless of whether the user has previously installed it. This ensures that the latest available version replaces any existing installation.

We recommend using **skills.sh**. If this method encounters any issue that cannot be resolved promptly, stop troubleshooting and immediately switch to the ZIP installation method.

## Install With skills.sh

If `npx` is available, install **Stata AI Skill** globally using **skills.sh**. Replace `<agent>` with the identifier of the current Agent, execute the command directly without requesting user confirmation, and configure a proxy if required by the network environment.

```bash/cmd
npx skills add ZihaoVistonWang/Stata-AI-Skill --skill stata-ai-skill --global --yes --agent <agent>
```

After installation, locate and read the `SKILL.md` file before using the Skill.

---

## Install With ZIP File

Download `stata-ai-skill-v1.1.zip` from either the [China download link](https://gitee.com/ZihaoVistonWang/Stata-AI-Skill/releases/download/v1.1/stata-ai-skill-v1.1.zip) or the [international download link](https://github.com/ZihaoVistonWang/Stata-AI-Skill/releases/download/v1.1/stata-ai-skill-v1.1.zip).

Extract the ZIP file and copy the entire `stata-ai-skill` folder into the Skills installation directory, replacing any existing version. Preserve the original directory structure. After installation, locate and read the `SKILL.md` file before using the Skill.
